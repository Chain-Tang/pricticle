# Particleify - 3D 粒子特效转换器 ✨

Particleify 是一个基于网页的互动工具，能够将你的静态图片 (PNG/JPG) 或者 3D 模型 (.glb / .gltf) 转换为令人惊叹的 3D 粒子特效。

本项目基于 Three.js 构建，所有的粒子动画和特效（如爆炸、漩涡、波浪等）均通过 WebGL 自定义着色器（Shaders）在 GPU 端高性能运行。

## 🌟 核心功能

*   **图片粒子化**：上传任何图片，自动提取非暗色背景的像素点，并映射到 3D 空间。
*   **3D 模型粒子化**：原生支持上传 `.glb` 或 `.gltf` 模型，自动采样模型表面并保留材质颜色。
*   **内置几何体**：内置正方体、球体、莫比乌斯环、等边锥体、DNA 螺旋、无限环等 3D 数学模型。
*   **丰富的视觉特效**：支持“默认漂浮”、“散射”、“爆炸”、“漩涡”、“脉冲”和“波浪”等多种动态效果。
*   **高度可定制**：支持实时调整粒子数量（10K ~ 200K）、粒子大小、去底色阈值，以及红/绿/蓝/白四色比例的混合分配。
*   **一键导出**：
    *   **导出 HTML**：将当前生成的粒子状态快照并打包成一个完全独立的 HTML 文件，无需服务器即可在任何浏览器中运行。
    *   **导出 MP4**：通过浏览器原生 MediaRecorder API 录制粒子旋转动画并导出为视频。

## 🚀 如何运行

本项目是一个纯前端的单文件应用（Single File Application）。

1. 克隆或下载本项目。
2. 使用任何本地 Web 服务器环境运行。例如使用 Python：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000/ImageToParticle.html` 即可使用。

> **注意**：由于涉及到本地文件上传和跨域策略（Canvas 读取图片数据），请务必通过 HTTP 服务器环境打开，不要直接双击 HTML 文件运行。
